# Bijli Live ⚡

**Pakistan's electric marketplace** — web + Android in one open repository.

Solar panels, inverters, batteries, EVs, and electrical products: connect buyers with trusted vendors.

## Download & use

```bash
git clone https://github.com/badrpk/bijli.git
cd bijli-live
```

### Web app

```bash
cd apps/web
npm install
npm run dev   # or: npm start
```

### Android app

```bash
cd apps/android
# open in Android Studio, or:
./gradlew assembleDebug
```
