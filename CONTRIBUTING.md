# Contributing to Bijli Live

Thank you for helping improve this project!

## Quick start for contributors

1. **Fork** the repo on GitHub  
2. **Clone** your fork  
3. Create a branch: `git checkout -b feature/your-idea`  
4. Make changes, keep commits clear  
5. Open a **Pull Request** against `main`

## Guidelines

- Keep each change focused (one feature or fix per PR)
- Document user-facing changes in README
- Prefer simple, readable code over cleverness
- Do not commit secrets, API keys, or `.env` files

## Local run

See README **Quick start**.

## Code of conduct

Be respectful. This is an open project for learning and real-world use in Pakistan and beyond.
